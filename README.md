# 🛒 AR SHOP · Tienda de Ropa con Realidad Aumentada

AR SHOP es una solución innovadora que permite visualizar productos en **Realidad Aumentada (AR)** antes de realizar la compra. El ecosistema incluye una aplicación móvil para usuarios y un panel administrativo web para la gestión total del inventario y pedidos.
